# 第一批真实模型接入实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为当前动态漫剧项目接入第一批真实模型能力，让剧本稿、角色图、场景图和语音不再依赖本地 mock。

**Architecture:** 文本层直接升级 `ScriptDraftService`，优先调用火山文本模型并映射为现有 `StoryboardDraft`。图片和语音继续走现有 provider + job 机制，在 `ProviderRegistry` 中新增 `ark_minimax` provider，并把远程结果下载到项目本地资产目录。

**Tech Stack:** FastAPI, httpx, Pydantic, Celery, 火山 Ark, MiniMax

---

### Task 2: 接入火山文本模型生成剧本稿

**Files:**

- Modify: `backend/app/services/project_service.py`

- [ ] **Step 1: 为文本模型增加调用层**

当模型调用异常或结构化结果不合法时，回退到现有规则拆解。
